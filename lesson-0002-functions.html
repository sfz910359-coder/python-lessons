② 定义函数：def

你的质数判断程序，用函数改写后是这样：

C++
bool is_prime(int n) {
    if (n < 2) return false;
    for (int i = 2; i*i <= n; i++) {
        if (n % i == 0) return false;
    }
    return true;
}
Python
def is_prime(n):
    if n < 2:
        return False
    for i in range(2, int(n ** 0.5) + 1):
        if n % i == 0:
            return False
    return True
C++Python说明

{ 花括号: + 缩进缩进就是作用域！
return false;return False大写 F，无分号
类型必须声明运行时推断鸭子类型 🦆
⚠️ 最大坑：缩进必须一致！ 4 空格或 Tab，混用直接 IndentationError。

③ 参数魔法 — Python 特有玩法

3.1 默认参数

C++
// C++ 也有默认参数
string greet(string name, string g="Hallo") {
    return g + ", " + name + "!";
}
Python
def greet(name, greeting="Hallo"):
    return f"{greeting}, {name}!"

print(greet("Felix"))          # Hallo, Felix!
print(greet("Felix", "Moin"))  # Moin, Felix!

注意 Python 的 f"{...}" — f-string，花括号里写变量名直接替换。比 C++ 的 + 拼接优雅一万倍 🎯

3.2 关键字参数（C++ 没有！）

def beschreibe(name, alter, stadt="Unbekannt"):
    return f"{name}, {alter} Jahre, aus {stadt}"

# 按顺序传：
print(beschreibe("Anna", 20, "Berlin"))

# 按名字传 — 顺序无所谓！
print(beschreibe(alter=20, name="Anna", stadt="Köln"))

# 混合也行：位置参数在前，关键字在后
print(beschreibe("Anna", alter=20))

关键字参数让函数调用自文档化 — 看到 alter=20 不用翻定义就知道什么意思。

④ 多返回值

C++
// 需要 struct / pair 或引用参数
struct Ergebnis {
    int summe;
    double durchschnitt;
    int maximum;
    bool alle_bestanden;
};
Python
def analysiere_noten(noten):
    s = sum(noten)
    avg = s / len(noten)
    mx = max(noten)
    ok = all(n >= 60 for n in noten)
    return s, avg, mx, ok

# 解包接收：
summe, avg, mx, ok = analysiere_noten([85, 92, 58, 73])
print(summe)  # 308
print(ok)     # False（58 < 60）

Python 一行 return a, b, c, d 就搞定了 — 本质是返回一个 元组（tuple），然后自动解包。

⑤ Lambda — 一行函数

# 普通函数写法
def quadrat(x):
    return x ** 2

# Lambda 一行版
quadrat = lambda x: x ** 2

# 真正有用的地方：排序
studenten = [("Anna", 92), ("Ben", 78), ("Clara", 95)]
studenten.sort(key=lambda s: s[1])  # 按分数排序
print(studenten)  # [('Ben', 78), ('Anna', 92), ('Clara', 95)]
语法含义
lambda x: x ** 2输入 x，返回 x²
lambda a, b: a + b输入 a, b，返回和
lambda s: s[1]取元组的第二个元素（索引 1）

Lambda 不能写复杂逻辑（没有 if/for），只能是一个表达式。C++ 里的匿名函数 [](int x){ return x*x; } 就是类似的东西。

⑦ 🎯 翻译练习

把下面的 C++ 函数翻译成 Python

功能：接收一个成绩列表，返回总分、平均分、最高分、是否全部及格（≥60）

// C++ 原版
struct Ergebnis {
    int summe;
    double durchschnitt;
    int maximum;
    bool alle_bestanden;
};

Ergebnis analysiere_noten(vector<int> noten) {
    Ergebnis e;
    e.summe = 0;
    e.maximum = noten[0];
    e.alle_bestanden = true;
    
    for (int n : noten) {
        e.summe += n;
        if (n > e.maximum) e.maximum = n;
        if (n < 60) e.alle_bestanden = false;
    }
    
    e.durchschnitt = (double)e.summe / noten.size();
    return e;
}

💡 提示（从易到难）

  1. sum(noten) — 直接求总和
  2. max(noten) — 直接取最大值
  3. len(noten) — 列表长度
  4. all(n >= 60 for n in noten) — 一行判断全部及格
  5. 用 return a, b, c, d 多返回值
