① Dateien öffnen — open()

Bisher hat dein Programm nur mit Dingen gearbeitet, die du direkt eingetippt hast. Aber richtige Programme lesen und speichern Daten in Dateien.

Eine Textdatei lesen:

# datei.txt enthält: "Hallo Felix!"
f = open("datei.txt", "r")     # "r" = read = lesen
inhalt = f.read()                # ganze Datei einlesen
print(inhalt)                   # Hallo Felix!
f.close()                       # Datei wieder zumachen!
ModusBedeutungWas passiert?
"r"read [读]Nur lesen. Datei muss existieren.
"w"write [写]Schreiben. Löscht alten Inhalt!
"a"append [追加]Anhängen. Altes bleibt, Neues kommt hinten dran.
🍪 Snack: Dateien sind einfach nur lange Strings auf der Festplatte. f.read() gibt dir den GESAMTEN Inhalt als EINEN String — mit allen Zeilenumbrüchen (\n).

② Dateien schreiben

f = open("notizen.txt", "w")   # "w" = write — überschreibt!
f.write("Heute Python gelernt.\n")
f.write("Morgen weiter.\n")
f.close()

Zeilenweise lesen:

f = open("notizen.txt", "r")
for zeile in f:
    print("→", zeile.strip())   # .strip() entfernt \n
f.close()

③ with — der Lebensretter

Das Problem: Du machst open(), dein Programm crasht, close() wird nie aufgerufen. → Datenverlust!
Die Lösung: with
with open("datei.txt", "r") as f:
    inhalt = f.read()
# Kein close() nötig! with macht das automatisch.

Merkregel: Immer with open(...) as f: benutzen.

④ try/except vertiefen

try:
    f = open("gibt_es_nicht.txt", "r")
    inhalt = f.read()
    zahl = int(inhalt)
except FileNotFoundError:
    print("❌ Datei existiert nicht!")
except ValueError:
    print("❌ Inhalt ist keine Zahl!")
except Exception as e:
    print("❌ Irgendwas anderes:", e)
else:
    print("✅ Alles gut! Zahl =", zahl)
finally:
    print("🧹 Aufräumen... (läuft IMMER)")
BlockWann läuft er?
try:Der gefährliche Code
except Fehlertyp:Nur wenn DIESER Fehler auftritt
else:Nur wenn KEIN Fehler passiert
finally:Läuft IMMER — egal ob Fehler oder nicht

⑤ Pfade verstehen

# Relativer Pfad — vom aktuellen Ordner aus
open("daten.txt")           # gleicher Ordner wie das Skript
# Absoluter Pfad — von der Wurzel
open("/Users/zsf/Desktop/daten.txt")  # Mac/Linux
🍪 Snack: Das Dateisystem ist ein riesiger Baum. / ist die Wurzel. /Users/zsf/Desktop heißt: "Geh von / zu Users, dann zu zsf, dann zu Desktop." Auf Windows: C:\ — gleiche Idee.

⑦ Aufgabe: Lern-Tagebuch

Schreib ein Programm, das dein persönliches Lerntagebuch führt:

  1. Fragt: "Was hast du heute gelernt?"
  2. Hängt die Antwort an lerntagebuch.txt an — mit Datum
  3. Fragt: "Soll ich dir zeigen, was du gelernt hast? (j/n)"
  4. Wenn "j" → zeigt alle bisherigen Einträge

Gerüst:

from datetime import datetime  # für das Datum
heute = datetime.now().strftime("%d.%m.%Y")  # z.B. "03.07.2026"
eintrag = input("Was hast du heute gelernt? ")
with open("lerntagebuch.txt", "a") as f:
    f.write(f"{heute}: {eintrag}\n")
💡 Tipp: datetime ist ein Modul (kommt in Lektion 5). datetime.now() = aktuelle Uhrzeit, .strftime("%d.%m.%Y") = deutsches Datum.
