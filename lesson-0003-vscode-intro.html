① VSCode 到底是什么？

很多人以为 VSCode 是「写 Python 的软件」—— 它远不止于此。

VSCode = 代码编辑器 + 轻量 IDE + 万能工具箱

它本身只是一个文本编辑器，但通过扩展（Extensions）可以变成：
• Python 开发环境
• 网页前端设计工具（HTML/CSS/JS）
• 数据分析平台（Jupyter Notebook）
• Markdown 写作工具
• Git 版本管理界面
• 甚至可以用来写小说、记笔记
工具相当于VSCode 能替代吗？
记事本纯文本编辑✅ 完全可以
Word排版文档⚠️ 不推荐（用 Markdown 可以）
PyCharmPython 专用 IDE✅ 装 Python 扩展后接近
终端.app命令行✅ 内置终端
Finder文件管理✅ 侧边栏文件浏览器
Sublime Text轻量编辑器✅ 完全替代
🧊 认知休息站
VSCode 用了 Electron 技术 —— 把网页技术（HTML/CSS/JS）打包成桌面应用。所以它的底层其实就是 Chrome 浏览器！这也是为什么它能做这么多事情：因为任何网页能做的事，VSCode 都能做。

③ 编辑区 · 核心工作区

这是你待得最久的地方。几个关键概念：

3.1 标签页（Tabs）

和浏览器一样，VSCode 可以同时打开多个文件，顶部有标签页切换。
Cmd + W 关闭当前文件 · Cmd + Shift + T 重新打开刚关的

3.2 语法高亮（Syntax Highlighting）

def analysiere(noten):       # ← 灰色：注释
    s = sum(noten)          # ← 紫色：关键字  青色：函数  绿色：字符串
    return s                 # 不同颜色 = 不同「身份」

不是装饰 — 它帮你一眼看清代码结构。哪里是函数、哪里是字符串、哪里是注释，全用颜色区分。

3.3 智能提示（IntelliSense）

写到一半会自动弹出补全建议：

print(|          ← 光标在这里，弹窗显示所有可用的函数/变量
→ 弹窗显示：print(value, sep, end, file, flush)
→ 按 Tab 或 Enter 补全

3.4 多光标编辑

按住 Alt（Mac 上是 ⌥ Option）然后鼠标点多个位置 → 同时编辑多处！

3.5 代码折叠

行号左边的小三角 ▸ → 点击折叠/展开整个函数。大文件必备。

📁 Explorer · 重要概念：工作区（Arbeitsbereich）

VSCode 不是「打开一个文件」— 它打开一个文件夹作为工作区。

所以你会看到整个文件夹的结构：子文件夹、所有文件。这和 Word 完全不同 — Word 只打开一个 .docx，VSCode 打开整个项目。

建议：把你所有 Python 练习放在一个文件夹里（比如 Programmieren lernen/），然后 Datei → Ordner öffnen 打开它。

⑤ 终端（Terminal）

底部面板里的终端，和 Mac 自带的「终端.app」是同一个东西。

为什么要在 VSCode 里用终端？

✅ 不用来回切换窗口
✅ 终端的当前目录自动 = 你打开的项目文件夹
✅ 报错时可以点击跳转到对应代码行
✅ 可以同时开多个终端（点 + 号）

常用终端命令：

# 运行 Python 文件
python3 uebung.py

# 安装 Python 包
pip3 install irgendein-paket

# 看当前文件夹有什么
ls

# 进入子文件夹
cd Programmieren\ lernen/
如果终端不见了：菜单 Ansicht → Terminal（查看 → 终端）
或者快捷键 Ctrl + `（反引号，Tab 上面那个键）

⑥ 命令面板 · 万能入口

这是 VSCode 最重要的快捷键，没有之一：

Cmd + Shift + P
德语名：Befehlspalette

按下去弹出一个搜索框，所有 VSCode 功能都在里面。你不需要记住菜单在哪 — 搜就完了：

搜这个效果
Sprache切换界面语言（中/德/英）
Theme换配色主题
Python: Interpreter选择 Python 版本
Terminal打开/新建终端
Einstellungen打开设置
Zoom放大/缩小界面
💡 Profi-Tipp
记不住按钮在哪？Cmd + Shift + P 搜一下。这是 VSCode 老手的肌肉记忆。

⑨ 不只是 Python — VSCode 还能做什么

🌐 网页开发

HTML、CSS、JavaScript 原生支持。装 Live Server 扩展后，保存即刷新浏览器预览。

你已经写过的 lesson-0002-functions.html 就是用 VSCode 编辑最合适的文件类型。

📝 写 Markdown

所有 .md 文件（比如你的学习笔记、项目文档）都是 Markdown 格式。
按 Cmd+Shift+V 可以实时预览排版效果。

📊 数据分析

装 Jupyter 扩展后，可以在 VSCode 里直接跑 Jupyter Notebook。
大学里处理数据、画图表、做实验报告都能用。

🔄 Git 版本管理

不需要学命令行 Git。左侧 🔗 图标点进去：
• 看改了什么
• 一键提交
• 同步到 GitHub
大学做小组项目时必备。

🌍 多语言支持

不只是 Python。VSCode 原生支持或通过扩展支持：
C/C++, Java, JavaScript, TypeScript, Go, Rust, Ruby, PHP, Swift, R, SQL, HTML/CSS, JSON, YAML, XML...

🎯 实操任务

打开 VSCode，完成以下 5 件事（按顺序）：

  1. Cmd+Shift+P → 输入 Theme → 换个你喜欢的配色
  2. Ctrl+` 打开终端 → 输入 python3 --version 回车
  3. 在左边 Explorer 右键 → Neue Datei → 创建 test.py
  4. 输入 print("Hallo VSCode!") → Cmd+S 保存
  5. 右键代码区 → Python ausführen → Python-Datei im Terminal ausführen

全部成功 = 你已掌握 VSCode 基本操作 ✅
